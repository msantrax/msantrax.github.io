ASVP Project

  • An instrument able to measure surface area of compounds by gas adsorption isotherms according with ISO 9277:2010. Uses the new Opus platform (O.S., Calc Server/ML and UI) to serve as analytical interface.

AutoDensity Project

  • A Pycnometer able to measure density on ores/cement by gas volumetry compliant to standard ASTM D-2856 and ASTM D-6683. Uses the Opus Netbeans based analytical platform and a stand alone (LCD) version

LGT8 Project

  • Retrofit a Logitech precision lap and polishing machine Series LP (to geological samples). Uses Android based Opus instrument interface platform

Blaine PP200 Project

  • Instrument able to measure powders' fineness by using the Blaine technique compliant to the standard ABNT 16372. Uses the new Opus Analytical Platform & O.S.

Centauro Project

  • Controlled atmosphere induction furnace to metal/ceramic materials capable of operating over 1500°C for extended periods of time.

Vehicle Dynamics Project

  • Retrofit a Jurid instrument used to capture a vehicle's sensor network data. Uses the Opus Netbeans based analytical platform

DGT100 Project

  • Upgrade a Microwave Digestion Device with a modern microcontroller and double the delivered power compared with similar devices

CCD Scan Research Project

  • A Python (React/Dash) research server and API provider. Hosted on a GCP Kubernetes cluster with an associated WEB server & load balancer (custom Jetty on Linux Debian)

Iris Project

  • Tasks to refurbish an Inductively Coupled Plasma Spectrometer (ICP) Iris model from Thermo Jarrel. Replace (and make compatible) his CCD Camera, and modify entrance optics (radial to axial)

Antares Project

  • Provide hardware, firmware, and analytical user interface to coordinate a pool of 10 instruments able to execute long-term Metallurgical Creep Tests compliant with the standard ISO 204 / ASTM E039

Medusa Project

  • Provide ways to intercept, filter, and transform analytical results from instruments unable to talk to enterprise database managers. Uses the Opus QT/C++ based analytical platform on Linux machines

MiddlestripB

  • Holds code that will interface user procedures with the automation (via attached ARM microcontroller) & the Python analytical calculation server used on the Instruments Platform

FXSupport

  • Support libraries and tools to the JavaFX user interface and activity manager used on the Analytical Instruments Platform

Virna7

  • Android (on tablets) User Interface and Control to Precision Geological Sample Preparation Device. Embedded Controler Hardware/Firmware/CAD/PCB files are under LGT8 callsign.

PP200

  • JavaFX - Stand Alone S.O. / Work Canvas / Window Manager & Analysis Instrument Platform (Github demo version)

ASVPServer1

  • Python research server to do Adsorption Isotherm Calculation and Strategics. Core and event gate between Jupyter/IPython Server and ASVP instrument controller

Temps1 Portal

Control-E

Microwave Organic Synthesis
